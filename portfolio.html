Using Python Turtle to Create Snake Game

In this project, we were asked to create an animation with the Turtle function in Python. Our group had designed an animation that drew inspiration from the classic snake game, where an object that is meant to resemble a snake navigates across a square map, eating apples and growing as it moves.

We utilized such functions as:

  • If conditional statements to check for object collisions with either the apples or the walls
  • While loops to condense snake movement into multiple iterations of a specific pattern
  • Write commands to enable the turtle object to display text

Creating "Tic-Tac-Blackout" with Python

Project 1.2.5 of PLTW asked that we design a game based off of our existing understanding of Python and whatever other resources we had at hand. Utilizing Python Turtle, our group designed a remix of tic-tac-toe: "Tic-Tac-Blackout." With the intent of incorporating a 'blind' twist on the traditional game, our group constructed a program that blacked out the screen and only lit back up upon clicking a section of the board and placing an 'x' or 'o.'

In order to build this game, we utilized such functions as:

  • Class/def functions to categorize multiple actions under singular functions that can be more easily called upon later within the code
  • Self (instances within a class; i.e. an accessible variable x within def function) parameter in order to create designated variables for callable functions
  • For loops in order to iterate through the board building sequence a set amount of times so that the board is arrange properly
  • .onclick Turtle function in order to create an interface that is directly responsive to player interaction
